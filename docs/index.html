Arquitectura del Sistema

Servidor OPC-UA

Espacio de direcciones modelado jerárquicamente. Emulación de sensores de temperatura y presión con lógica asíncrona.

FastAPI Bridge

Middleware asíncrono que convierte el tráfico binario OPC-UA en flujos de datos JSON vía WebSockets para la web.

HMI Dashboard

Interfaz enriquecida con Chart.js y optimizada para pantallas industriales táctiles o centros de control.
